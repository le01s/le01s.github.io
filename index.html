About Me

Hi! I'm an experienced backend developer with a strong passion for building powerful, scalable, and secure web applications. My expertise lies in crafting robust and efficient server-side solutions that ensure fast and stable performance for websites and applications.

I specialize in backend development using Python, with a keen interest in web technologies. My primary focus is on information security, including both black-box and white-box testing methods. I also develop scripts for automating various processes, particularly in the field of information security. Additionally, I explore the development of API applications and working with microservice architectures.

What I Do:

  • Develop and maintain the server-side of web applications.
  • Work with databases (SQL and NoSQL) for data storage and processing.
  • Optimize application performance to ensure fast operation even under heavy load.
  • Implement security features, including authentication, authorization, and data protection.
  • Integrate third-party APIs and services to extend application functionality.

My Tech Stack:

  • Programming Languages: Python.
  • Frameworks: FastAPI and many others libs and integrations.
  • Databases: PostgreSQL, MySQL, MongoDB.
  • Tools: Docker, Kubernetes, Git.
  • Queues and Caching: RabbitMQ.
  • Operation systems: Linux (debuan, ubuntu, centos).
  • Network knowledges: TCP/IP, OSI, VPN, etc.

My philosophy:

I believe that the backend is not just a "black box" hidden behind the frontend. It's the heart of the application that ensures its functionality, security, and scalability. My goal is to create solutions that not only work but also exceed user expectations.
